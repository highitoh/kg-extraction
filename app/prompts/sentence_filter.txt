入力された文書を行単位で解析し、各行がどの[[記述分類]]に属するかを判定してください。時間がかかっても良いので、一回で処理して下さい。入力処理の都合上、改行位置が直感と異なる場合がありますが、気にしないでください。
[[記述分類]]は優先度が高い順に並べています。複数当てはまる
同一分類が連続する場合は1つにまとめ、開始行と終了行を指定してください。
結果は必ず[[出力形式]]に従って返してください。

[[記述分類]]
- 表紙：文書のタイトルや著者、作成日付を示す記述
- 目次：文書の目次を示す記述
- 章タイトル：文章の章節項を示す記述
- 表：表を示す記述
- メモ書き：情報を忘れないために書かれた洗練されていない一時的な記述
- 文章：情報を伝えるために書かれた体系立った文。
- その他：上記以外の記述

[[出力形式]]
- 出力は1つのJSON配列のみとし、余分な文字を含めないこと。  
- スキーマは以下に従うこと：
{
  "items":
  [
    {
      "content": "記述分類",
      "startLine": "開始行数(1始まり)",
      "endLine": "W"
    }
  ]
}
