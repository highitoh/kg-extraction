以下の《原文チャンク》から、value_analysis の情報抽出項目をポリシーに従って抽出してください。
同一文意の範囲での軽い言い換えは可。ただし意味の改変は禁止です。
各フィールドは名詞句で簡潔にし、ポリシー範囲の文字数に収めてください。


【ポリシー】
- ステークホルダ: 2～10字。個人名は避け、役割名を優先。
- 価値: 4～30字。『〜の向上』『効率化』など名詞句で簡潔に。

【抽出項目】
- class: クラスの種類
- label: 抽出したインスタンスの名前
- source: 抽出元の文章

【特記事項】
- ステークホルダは個人名を避け役割名を優先。
- 不明な項目はキーごと省略（空文字・null は使わない）。
- JSON 以外（説明文・改行やコードブロック）は一切出力しない。

【出力JSON仕様（例）】
{
  "classes": [
    {"class":"ステークホルダ", "label":"店舗スタッフ", "source":"店舗スタッフの業務効率を向上させる"},
    {"class":"価値", "label":"待ち時間短縮", "source":"顧客の待ち時間短縮を実現"}
  ]
}
