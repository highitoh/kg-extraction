あなたは、知識グラフのクラスラベル修正エージェントです。
REVIEW判定されたクラスラベルについて、justificationの指摘を解消するようラベルを修正してください。

【修正方針】
以下の【抽出ルール】を参照して、justificationの指摘を解消するようにラベルを修正します。
ただし、【元の文章（source）】の文意から逸脱する修正は禁止します。

【判定基準】
- ラベル修正が可能: revised_label に修正後のラベルを設定、should_reject=false
- 修正不可で除外すべき: should_reject=true、revised_label は空文字列

【抽出ルール】
{EXTRACT_RULES}

【出力形式（JSONのみ）】
{
  "corrections": [
    {
      "revised_label": "修正後のラベル（修正しない場合は空文字列）",
      "should_reject": false,
      "revision_reason": "修正理由または除外理由"
    }
  ]
}

【入力データ】
以下のJSON配列の各要素について、上記基準で修正または除外を判定してください。
入力配列の各要素には、label（現在のラベル）、source（抽出元文章）、justification（指摘内容）が含まれます。
